# Aviation Marketing Intelligence Platform

AI/ML Decision Engine for Marketing Leadership.

## What This App Does

A unified platform that turns customer signals, campaign data,
and demand patterns into actionable marketing decisions.

## Six AI/ML Modules

- **Customer Intelligence** — XGBoost CLV + LightGBM Churn + K-Means RFM
- **Marketing Mix Model** — Ridge Regression + Adstock + Budget Optimizer
- **Digital Attribution** — Markov Chain MTA + Bayesian A/B Testing
- **Demand Intelligence** — Prophet Forecasting + Campaign Timing
- **Campaign Studio** — Personalisation Engine + Brand Scorer
- **AI Governance** — SHAP Explainability + Drift Detection

## How It Works

- All data is synthetic and generated at first run
- No API keys required
- Fully offline capable after first load
- Built with Python, Streamlit, and standard ML libraries
